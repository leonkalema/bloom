What is a Bloom Filter?

A Bloom Filter is a space-efficient probabilistic data structure that tests whether an element is in a set. It can have false positives but never false negatives.

Key Benefit: Uses only a few KB to track millions of items with 99%+ accuracy

Perfect for:

  • ‚Ä¢ Caching: Check if data exists before expensive lookups
  • ‚Ä¢ Deduplication: Filter duplicate emails, URLs, or records
  • ‚Ä¢ Security: Block malicious IPs or domains
  • ‚Ä¢ Databases: Avoid disk reads for non-existent keys

Enterprise Features

Auto-scaling Dynamically resize as data grows
TTL Expiration Items expire after configurable time
Set Operations Union and intersection of filters
Memory Pool Efficient filter reuse and management
Compression RLE compression for sparse data

Quick Start

Install via NPM

npm install adaptive-bloom-filter

Basic Usage

const { AdaptiveBloomFilter } = require('adaptive-bloom-filter');
const filter = new AdaptiveBloomFilter(10000, 0.01);
filter.add('user-123');
console.log(filter.check('user-123')); // true
